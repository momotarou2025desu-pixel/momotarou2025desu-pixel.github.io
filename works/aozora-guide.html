AozoraPC は、青空文庫の作品を取り込んで、見やすい読書モードで読める Windows アプリです。起動直後から流れがわかり、作品選択から読書モードへの移動もかんたんです。

  • ✓

  • ✓
    ZIP 展開後にすぐ使えます。

Features

ひと目でわかる、AozoraPC の流れ

起動直後の案内から、作品選択、読書モード、オートスクロールまでを短く整理しました。

起動直後から流れがわかる

起動すると案内が表示されるので、初めてでも流れがわかります。

人気作品からすぐ選べる

青空文庫へ進むと、人気作品やジャンル別一覧から名作をすぐ選べます。

ワンクリックで読書モードへ

作品ページを開いたら、点滅する「ここを取り込む」を押すだけで読書モードに切り替わります。

右クリックでオートスクロール

読書モードでは右クリックでオートスクロールできるので、手を止めてゆっくり読み進められます。

How to use

起動手順

AozoraPC の起動手順を 5 ステップでまとめました。最初に Vector からダウンロードします。

1

Vector から AozoraPC.zip をダウンロードしてデスクトップに置く

まず Vector から AozoraPC.zip をダウンロードし、わかりやすい場所に置きます。

2

右クリックして「すべて展開」を選ぶ

ZIP ファイルを展開して、中のファイルを使える状態にします。

3

展開したフォルダを開き、AozoraPC をダブルクリックする

フォルダをたどって AozoraPC を開き、起動します。

4

Windows の保護画面が出たら「詳細情報」を選ぶ

Windows の表示に従って、実行できる画面へ進みます。

5

「実行」を押して起動する

そのまま起動すれば、すぐに青空文庫の読書を始められます。

作品ページで「ここを取り込む」を押すと、大きい文字の読書モードへ切り替わります。読書中は右クリックでオートスクロールできます。
