No.1 完全無料ホームページ作成 GitHub Pages 最短公開

ホームページを、
最短で公開する。

GitHub に HTML を入れて、公開URLを確認するまでの流れだけを残した案内ページです。上から順に進めれば、そのまま公開確認まで進めます。

公開手順を見る 実例を見る

Flow

公開手順

登録から公開URL確認まで、必要な操作だけを上から順に並べています。

1

GitHub に登録してログイン

最初に GitHub のアカウントへ入ります。ここから公開作業を始めます。

2

Create repository で保存場所を作る

HTML を置く保存場所を先に作ります。公開用の土台はここです。

3

Add file → Upload files でホームページの HTML を入れる

作った保存場所に、公開したい HTML をそのまま入れます。

4

Commit changes で確定する

アップロードした内容をここで確定します。反映の入口になる操作です。

5

Settings → Pages で公開URLを確認する

公開後に使うURLは Pages 画面で見ます。公開確認もここが基準です。

6

わからなければ、Alt+PrintScreenで今の画面をコピーして、AIさんにCtrl+Vで貼り付ける

迷った画面をそのまま見せる形で進めれば、操作を絞って確認できます。

最初は公開まで通すことを優先し、細かな整形や追加機能は公開確認のあとに進めると安定します。
