完全無料ホームページ作成

GitHubにホームページを入れて公開するまでの流れを、最短でまとめています。

公開までの流れ

  1. 1GitHubに登録してログイン
  2. 2Create repositoryで保存場所を作る
  3. 3Add file → Upload filesでホームページのHTMLを入れる
  4. 4Commit changesで確定する
  5. 5Settings → Pagesで公開URLを確認する
  6. 6わからなければ、Alt+PrintScreenで今の画面をコピーして、AIさんにCtrl+Vで貼り付ける
